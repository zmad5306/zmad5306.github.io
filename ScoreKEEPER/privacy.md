# Privacy Policy for ScoreKEEPER

**Last updated:** December 2025

ScoreKEEPER is a simple, offline score-tracking application designed for card games, board games, and other tabletop games.

## Information Collection and Use

ScoreKEEPER does **not collect**, **store**, or **transmit** any personal information.

- No user accounts  
- No analytics  
- No advertising  
- No tracking technologies  
- No network communication

All scores and game data remain entirely on the user's device.

## Third-Party Services

ScoreKEEPER does **not** use any third-party services that collect user data.

## Data Security

Because ScoreKEEPER does not collect or transmit any data, no personal data is stored or processed by the developer.

## Children's Privacy

ScoreKEEPER does not knowingly collect any personal information from children or users of any age.

## Changes to This Privacy Policy

If ScoreKEEPER's functionality changes in a way that affects user privacy, this policy will be updated accordingly.
